# CrazyGames Technical Requirements

## 📋 技术要求概述

本文档描述了在CrazyGames平台发布游戏的技术要求和最佳实践。

## 🎮 游戏格式要求

### **支持的游戏格式**
- **HTML5游戏**: 使用WebGL、Canvas或DOM渲染
- **Unity游戏**: WebGL构建版本
- **Construct游戏**: HTML5导出版本
- **其他引擎**: 支持Web导出的游戏引擎

### **文件大小限制**
- **初始加载**: 建议不超过20MB
- **总大小**: 建议不超过100MB
- **资源优化**: 使用压缩和优化技术减少文件大小

### **性能要求**
- **加载时间**: 首次加载不超过10秒
- **帧率**: 稳定30FPS，目标60FPS
- **内存使用**: 合理的内存占用，避免内存泄漏
- **CPU使用**: 优化CPU使用，避免过度消耗

## 🌐 浏览器兼容性

### **支持的浏览器**
- **Chrome**: 最新版本
- **Firefox**: 最新版本
- **Safari**: 最新版本
- **Edge**: 最新版本
- **移动浏览器**: iOS Safari、Chrome Mobile

### **Web标准支持**
- **HTML5**: 完整的HTML5支持
- **CSS3**: 现代CSS特性
- **JavaScript**: ES6+特性
- **WebGL**: WebGL 1.0/2.0支持
- **WebAudio**: 音频API支持

## 📱 移动设备支持

### **响应式设计**
- **自适应布局**: 支持不同屏幕尺寸
- **触摸控制**: 优化的触摸输入
- **设备方向**: 支持横屏和竖屏
- **性能优化**: 移动设备性能优化

### **移动端特性**
- **触摸手势**: 支持常见触摸手势
- **虚拟按键**: 移动端友好的UI控件
- **电池优化**: 避免过度消耗电池
- **网络优化**: 适应移动网络条件

## 🔧 游戏集成要求

### **CrazyGames SDK集成**
```javascript
// CrazyGames SDK基本集成
import { CrazySDK } from 'crazysdk';

const sdk = new CrazySDK();

// 游戏开始时初始化
sdk.init();

// 显示广告
sdk.ad.showInterstitial();

// 游戏结束
sdk.game.gameplayStop();
```

### **必需的API调用**
- **sdk.init()**: 游戏初始化
- **sdk.game.gameplayStart()**: 游戏开始
- **sdk.game.gameplayStop()**: 游戏结束
- **sdk.ad.showInterstitial()**: 显示插页广告
- **sdk.ad.showRewarded()**: 显示激励广告

### **广告集成**
- **插页广告**: 在合适的时机显示
- **激励广告**: 提供奖励机制
- **横幅广告**: 可选的横幅广告
- **广告频率**: 避免过度显示广告
