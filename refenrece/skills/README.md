# Skills Directory

This directory contains the original skill files that have been integrated into Claude Code Game Studios agents.

## Structure

```
skills/
├── godot/                    # Godot-Claude-Skills-main
├── cocos-creator/            # cocos-creator-dev-skill-main
├── narrative-game/           # narrative-game-skill-main
└── unity-skills/             # OH-Unity-GameDev-Skills-main
```

## Purpose

- **Version Control**: Keep original skill files for reference
- **Documentation**: Provide detailed documentation for agents
- **Traceability**: Link agents back to their source skills
- **Backup**: Preserve original skill implementations

## Integration Status

- ✅ Godot-Claude-Skills-main → Godot Full-Stack Specialist
- ✅ cocos-creator-dev-skill-main → Cocos Creator Specialist
- ✅ narrative-game-skill-main → Narrative Game Generator
- ✅ OH-Unity-GameDev-Skills-main → Unity Development Specialist + Animation Specialists

## Usage

Each skill directory contains:
- `SKILL.md` - Main skill file
- `references/` - Supporting documentation
- `examples/` - Usage examples (if available)

These files serve as the authoritative source for the corresponding agents in the `../agents/` directory.
