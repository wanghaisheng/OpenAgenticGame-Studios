## Core Loop Framework

### 1. 循环结构设计
```
标准核心循环结构:
动作 (Action) → 奖励 (Reward) → 进度 (Progress) → 新动作
├── 动作设计
│   ├── 主要动作 (核心玩法)
│   ├── 辅助动作 (支持性玩法)
│   └── 社交动作 (玩家互动)
├── 奖励设计
│   ├── 内在奖励 (成就感、掌控感)
│   ├── 外在奖励 (物品、货币)
│   └── 社交奖励 (认可、地位)
└── 进度设计
    ├── 技能进度 (能力提升)
    ├── 内容进度 (新内容解锁)
    └── 社交进度 (地位提升)
```

### 2. 动机心理学应用
- **自主性需求** - 玩家选择和控制感
- **胜任感需求** - 技能掌握和挑战克服
- **关系需求** - 社交连接和归属感
- **目的需求** - 意义感和目标追求

### 3. 奖励系统设计
```
奖励类型矩阵:
├── 即时奖励 (秒级)
│   ├── 视觉反馈 (特效、动画)
│   ├── 音频反馈 (音效、音乐)
│   └── 触觉反馈 (震动、响应)
├── 短期奖励 (分钟级)
│   ├── 物品奖励 (道具、装备)
│   ├── 货币奖励 (金币、钻石)
│   └── 经验奖励 (经验值、等级)
├── 中期奖励 (小时级)
│   ├── 成就奖励 (成就、徽章)
│   ├── 解锁奖励 (新内容、新功能)
│   └── 社交奖励 (排名、认可)
└── 长期奖励 (天/周级)
    ├── 里程碑奖励 (重大成就)
    ├── 季节奖励 (限定内容)
    └── 社交地位奖励 (声望、地位)
```
