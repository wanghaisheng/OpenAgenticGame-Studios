## 🎯 Skill Selection Guide

### **By Team Size**
- **Small Teams** (1-5 people): Use core skills + essential team skills
- **Medium Teams** (6-15 people): Add specialized skills + engine-specific skills
- **Large Teams** (16+ people): Use all skills + advanced workflow skills

### **By Project Phase**
- **Pre-Production**: Planning, design, and setup skills
- **Production**: Development, testing, and iteration skills
- **Post-Production**: Release, launch, and support skills

### **By Complexity**
- **Simple Projects**: Core skills only
- **Complex Projects**: Core + team + engine skills
- **Enterprise Projects**: All skills + advanced workflows

## 🔄 Skill Integration

### **With Agent System**
- Each skill integrates with relevant team agents
- Skills can be invoked directly or through agents
- Team agents can recommend appropriate skills

### **With Quality Gates**
- Skills include built-in quality gate validation
- Quality gates ensure skill output meets standards
- Failed gates trigger escalation procedures

### **With Documentation**
- Skills generate standardized documentation
- Documentation follows team-specific templates
- Documentation is automatically version controlled
